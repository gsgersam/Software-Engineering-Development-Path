# ft_irc Flowchart (ASCII style)

## 1) End-to-end server flow

```text
+-------------------------------+
| Start: ./ircserv <p> <pass>   |
+-------------------------------+
               |
               v
+-------------------------------+
| Validate args + parsePort     |
+-------------------------------+
        | valid                 | invalid
        v                       v
+-----------------------+   +---------------------+
| Register SIGINT/TERM  |   | Print error + exit  |
+-----------------------+   +---------------------+
               |
               v
+-------------------------------+
| Create Server(port, password) |
+-------------------------------+
               |
               v
+-------------------------------+
| setupListenSocket()           |
| socket/bind/listen/nonblock   |
+-------------------------------+
               |
               v
+-------------------------------+
| addListenFdToPoll()           |
+-------------------------------+
               |
               v
+-------------------------------+
| run(): poll loop              |
+-------------------------------+
               |
               v
        +-------------------+
        | poll() has events?|
        +-------------------+
          | yes         | no/EINTR
          v             v
+------------------+   +------------------+
| iterate _pfds    |   | continue loop    |
+------------------+   +------------------+
          |
          v
+-------------------------+
| fd == listenFd ?        |
+-------------------------+
     | yes                      | no
     v                          v
+--------------------------+   +---------------------------+
| acceptNewClients()       |   | poll error/hup/nval ?     |
| accept loop until EAGAIN |   +---------------------------+
+--------------------------+        | yes           | no
     |                               v              v
     |                        +----------------+  +------------------+
     |                        | disconnect(fd) |  | POLLIN on fd ?   |
     |                        +----------------+  +------------------+
     |                                              | yes       | no
     |                                              v           v
     |                                    +----------------+  +------------------+
     |                                    | handleRead(fd) |  | POLLOUT on fd ?  |
     |                                    +----------------+  +------------------+
     |                                              |             | yes      | no
     |                                              v             v          v
     |                                    +-----------------+ +----------------+
     |                                    | parse/dispatch  | | handleWrite(fd)|
     |                                    +-----------------+ +----------------+
     |                                              |             |
     |                                              +------->-----+
     |                                                        |
     +--------------------------------------------------------+
                                                              v
                                                +---------------------------+
                                                | checkTimeouts()           |
                                                +---------------------------+
                                                              |
                                                              v
                                                +---------------------------+
                                                | g_serverRunning == false? |
                                                +---------------------------+
                                                   | yes             | no
                                                   v                 v
                                      +-----------------------+   (loop)
                                      | Exit loop, cleanup    |
                                      | close sockets, return |
                                      +-----------------------+
```

```text
+------------------+
| recv(fd, bytes)  |

          |
          v
+---------------------------+

+---------------------------+
          |
          v
+---------------------------+

+---------------------------+
          |
          v
+---------------------------+

+---------------------------+
          |
          v
+---------------------------+

+---------------------------+
          |
          v
+---------------------------+
| Dispatcher::handle()      |
+---------------------------+
          |
          v
+---------------------------+
| Handler (JOIN/KICK/...)   |
| validates + mutates state |
+---------------------------+
          |
          v
+---------------------------+
| queueToClient()/broadcast |
+---------------------------+
          |
          v
+---------------------------+

+---------------------------+
          |
          v
+---------------------------+
| send() in handleWrite()   |
+---------------------------+
```

## 3) Command routing and registration gate

```text
+------------------------------+
| Command parsed from client   |
+------------------------------+
               |
               v
+------------------------------+
| client.registered() ?        |
+------------------------------+
      | yes                          | no
      v                              v
+----------------------------+   +------------------------------+
| dispatch by command name   |   | allow only PASS/NICK/USER/  |
+----------------------------+   | QUIT                         |
      |                         +------------------------------+
      |                                | allowed       | blocked
      |                                v               v
      |                     +--------------------+  +------------------+
      |                     | run reg handler    |  | ERR 451          |
      |                     +--------------------+  +------------------+
      |                                |
      |                                v
      |                     +-----------------------------+
      |                     | PASS+NICK+USER complete ?   |
      |                     +-----------------------------+
      |                          | yes         | no
      |                          v             v
      |               +------------------+   (wait next)
      |               | set registered   |
      |               | send welcome     |
      |               +------------------+
      v
+---------------------------------------------------------------+
| Known command?                                                |
| PASS NICK USER PING QUIT PRIVMSG JOIN KICK INVITE TOPIC MODE |
| PART LIST NAMES                                               |
+---------------------------------------------------------------+
      | yes                                 | no
      v                                     v
+------------------------------+      +------------------+
| Execute handler              |      | ERR 421 unknown  |
+------------------------------+      +------------------+
```

## 4) Disconnect and cleanup flow

```text
+-----------------------------+
| Disconnect trigger          |
| QUIT / EOF / HUP / timeout  |
+-----------------------------+
              |
              v
+-----------------------------+
| requestDisconnect(fd, why)  |
+-----------------------------+
              |
              v
+-----------------------------+
| outBuffer empty ?           |
+-----------------------------+
      | yes                          | no
      v                              v
+--------------------------+   +----------------------+
| disconnectClient(fd)     |   | keep POLLOUT flush   |
+--------------------------+   +----------------------+
      |                              |
      +---------------<--------------+
              |
              v
+-----------------------------+
| notifyClientExit(QUIT msg)  |
+-----------------------------+
              |
              v
+-----------------------------+
| remove from all channels    |
+-----------------------------+
              |
              v
+-----------------------------+
| channel empty ?             |
+-----------------------------+
      | yes                          | no
      v                              v
+----------------------+      +---------------------------+
| delete channel       |      | ensureChannelHasOperator  |
+----------------------+      +---------------------------+
              \                  /
               \                /
                v              v
              +-----------------------------+
              | unbindNick + remove from    |
              | pollfd + close(fd) + erase  |
              +-----------------------------+
```
